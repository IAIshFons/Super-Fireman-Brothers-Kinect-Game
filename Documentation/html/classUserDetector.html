Detailed Description

This class handles the functions and methods of the user detection/handling for the application.

In this class we implement the functions that are necessary for user handling and detection. This functions are those referents to callbacks for user actions (new user, lost users, calibration, pose detection), specifications about tracked users and current transformations status.
