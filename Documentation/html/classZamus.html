Detailed Description

This class hanldes all aspects about the detection and information referred to the Zamus players.

This class handles the detection of the pose to transform in Zamus and other things like the Buster status and the shoots made by the users.

Specifies transformation stages.

  • T_STAGE_1 the transformation is less than 35%
  • T_STAGE_2 the transformation is less than 70%
  • T_STAGE_3 the transformation is less than 100%
