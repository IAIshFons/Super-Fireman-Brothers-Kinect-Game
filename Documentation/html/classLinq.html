Detailed Description

This class hanldes all aspects about the detection and information referred to the Linq players.

This class handles the detection of the pose to transform in Linq and other things like the pose to spawn ice and the ice spawned by the users.

Member Enumeration Documentation

enum Linq::IceRodStates

Ice rod stages.

  • DEACTIVATED the user can't shoot.
  • ACTIVATED the user can shoot.
enum Linq::lStages

Specifies transformation stages.

  • T_STAGE_1 the right arm of the user is transformed.
  • T_STAGE_2 the right and left arm of the user is transformed.
  • TRANSFORMED the user is transformed completly.
