Detailed Description

This class handles the drawing of the users when they are not transfomated in some character.

This class draws a stick figure over the users when they are been tracked but not transformated. Depending on the state of the user, it draws some parts of the characters models.

When the players are not tracked, this draws a diamond in their center of mass.
